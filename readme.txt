== Description ==

HivePress Authentication is an extension for [HivePress](https://wordpress.org/plugins/hivepress/) plugin. It allows users to sign in via third-party services.

[Demo](https://demo.hivepress.io/) | [Docs](https://hivepress.io/docs/extensions/authentication/) | [Support](https://hivepress.io/support/forum/extensions/authentication/)

You can enable authentication methods in HivePress/Settings/Integrations section of your WordPress dashboard. Currently the following authentication methods are available:

* Facebook - Register a Facebook app by following [these instructions](https://developers.facebook.com/docs/apps/#register), then enable Facebook Login for this app and paste the app ID in HivePress/Settings/Integrations section of your WordPress dashboard. You can check Facebook API terms of service [here](https://developers.facebook.com/policy/).
* Google - Create and configure a new Google API project by following [these instructions](https://developers.google.com/identity/sign-in/web/sign-in), then paste the client ID in HivePress/Settings/Integrations section of your WordPress dashboard. You can check Google API terms of service [here](https://developers.google.com/terms).
